Lista de Contatos

Sistema simples de cadastro de contatos desenvolvido como atividade acadêmica. O projeto permite adicionar, listar e excluir contatos utilizando HTML, CSS e JavaScript, com armazenamento no navegador.

Funcionalidades

Cadastro de contatos

Exibição automática na lista

Exclusão de contatos

Armazenamento utilizando localStorage

Interface simples e organizada

Tecnologias Utilizadas

HTML5

CSS3

JavaScript

Git e GitHub

Como Executar o Projeto

Clone o repositório

Abra a pasta do projeto

Execute o arquivo index.html no navegador

Objetivo do Projeto

Este projeto foi desenvolvido como prática para manipulação do DOM, uso de eventos em JavaScript, armazenamento de dados no navegador e organização da estrutura de um projeto web.
